数组的常见操作（2）

2017-07-04

本篇主要介绍数组的其他常见方法，主要从是否对数组发生改动的角度进行分析。mdn上将数组的方法分为mutator methods/accessor methods/itration methods。itration methods在上一篇中有简要介绍，这一篇主要介绍前两种。

mutator methods，对数组进行修改:

  • push/pop/shift/unshift
  • splice：该方法需要特别注意！既可以向数组中添加元素，也可删除元素。

例子如下：

  • reverse
  • sort
    此外，还包括copyWithin、fill，我第一次见到这两种方法

常用的Accessor methods，不对数组进行修改

  • concat，对数组进行合并

  • indexOf/lastIndexOf

  • join：Joins all elements of an array into a string.
  • slice：shallow copy

slice方法还经常用于将类数组的对象或集合转为新数组，比如Array.prototype.slice.call(arguments)

  • 上一篇中提及的forEach/filter/map/reduce均不对数组进行修改

slice/splice方法有时会弄混，至于它们的参数的具体情况（负数等），使用时再查阅文档吧。
在JavaScript中，strings和numbers都是immutable的。最近才看到mutable vs immutable，所以对其应用性等还不熟悉。等以后深入了解后再细谈。
